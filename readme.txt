# Term

Repository
Branch
Conflict

# Command
- git init // dự án có thể sử dụng được git sau khi đánh lệnh này, kiểu khởi tạo
- git status
- git add // chuẩn bị lưu lại trạng thái tại thời điểm hiện tại của dự án
- git reset // đổi ý chưa muốn lưu lại ở trạng thái hiện tại
- git commit
- git log
- git log --oneline // check the commit history
- git checkout {branch name}
- git branch
- git checkout -b {branch name} // to create a new branch
- git merge {branch name}
- git branch -d {branch name} // to delete the branch
